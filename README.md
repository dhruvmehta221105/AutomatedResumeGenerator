A desktop-based Resume Generator application built using Python, Tkinter, and ReportLab.
This project allows users to input personal and professional details, upload a profile image, and automatically generate a well-formatted PDF resume.

1. Personal details input (Name, Contact, Email, Address, etc.)

2. Education, Experience, Skills, Achievements, Certificates

3. Upload and preview profile image

4. Automatically generates a PDF resume

5. Simple and user-friendly GUI

6. Saves resume locally and opens it instantly

Tech Stack - Python
Tkinter – GUI development
Pillow (PIL) – Image handling
ReportLab – PDF generation
OS module – File handling
